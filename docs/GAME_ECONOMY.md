# Game Economy

## XP

- Correct answer: 10 XP
- Correct with hint: 6 XP
- Incorrect then corrected: 7 XP
- Quest completion: 25 XP
- Boss completion: 50 XP
- Daily comeback bonus: 20 XP

## Coins

- Correct answer: 5 coins
- Correct with hint: 3 coins
- Incorrect then corrected: 4 coins
- Quest completion: 20 coins
- Perfect quest bonus: 25 coins

## Leveling

`xpRequiredForLevel(n) = 100 + (n - 1) * 50`

## Forge Upgrades

- Apprentice Anvil: 50 coins, max level 3.
- Fraction Furnace: 75 coins, max level 3.
- Decimal Crystal Lamp: 100 coins, max level 3.
